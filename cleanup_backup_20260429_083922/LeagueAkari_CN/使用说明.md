# LeagueAkari - LOL数据助手 使用说明

## 📦 已创建的文件

### 核心功能文件
1. **`riot_api_client.py`** - Riot API客户端
   - 实现功能1和功能2的核心
   - 支持查询个人信息和他人信息
   - 支持排位、英雄熟练度、比赛历史查询

2. **`data_spider.py`** - 数据爬取模块（已更新）
   - ✅ 功能1: 检测本地LOL客户端并返回个人信息
   - ✅ 功能2: 通过玩家名称和ID查询他人信息
   - ✅ 功能3: OP.GG风格英雄榜单（分路排名、胜率、出装、符文）
   - ✅ 功能4: 更新按钮，点击后自动爬取最新英雄信息

3. **`ui_window_new.py`** - 新的图形界面
   - 标签页1: 个人信息（功能1）
   - 标签页2: 查询玩家（功能2）
   - 标签页3: 英雄榜单（功能3）
   - 标签页4: 设置/更新（功能4）

4. **`main_new.py`** - 新的主程序入口
   - 整合所有4个功能
   - 自动检测并加载Riot API配置

### 启动脚本
5. **`run.py`** - Python启动器（推荐）
   - 自动检查依赖
   - 测试所有功能
   - 可选择是否启动图形界面

6. **`启动新版.bat`** - Windows批处理启动脚本
   - 双击即可运行
   - 自动安装依赖

## 📋 4个功能详解

### ✅ 功能1：检测本地LOL客户端并返回个人信息
**位置**: 标签页1 "个人信息"

**功能**:
- 自动检测LOL客户端是否运行
- 显示召唤师名称、等级
- 显示排位信息（单双排、灵活排位）
- 显示英雄熟练度
- 显示最近比赛记录

**使用方法**:
1. 启动LOL客户端
2. 点击 "🔄 刷新个人信息" 按钮
3. 查看个人信息

**需要配置**:
- 无需配置（使用LCU本地接口）
- 如需更详细数据，可配置Riot API密钥

---

### ✅ 功能2：通过玩家名称和ID查询他人信息
**位置**: 标签页2 "查询玩家"

**功能**:
- 通过召唤师名称查询
- 显示排位信息
- 显示英雄熟练度TOP5
- 显示最近5场比赛记录

**使用方法**:
1. 在搜索框输入召唤师名称（如 "Faker"）
2. 点击 "搜索" 按钮
3. 查看玩家信息

**需要配置**:
- 必须配置Riot API密钥
- 获取方式: https://developer.riotgames.com/

**配置方法**:
1. 访问 https://developer.riotgames.com/
2. 注册账号并获取API密钥
3. 在 "设置/更新" 标签页输入API密钥
4. 选择服务器区域（韩服/日服/北美等）
5. 点击 "保存API设置"

---

### ✅ 功能3：OP.GG风格英雄榜单
**位置**: 标签页3 "英雄榜单"

**功能**:
- 显示所有英雄的胜率、选取率、禁用率
- 支持分路筛选（全部/TOP/JUNGLE/MIDDLE/BOTTOM/UTILITY）
- 显示英雄排名（TOP 20）
- 支持查看英雄出装和符文推荐

**使用方法**:
1. 选择分路（默认"全部"）
2. 查看英雄排名列表
3. 点击 "🔄 刷新" 按钮更新数据

**数据源**:
- 英雄列表: 腾讯CDN（国内快速访问）
- 装备数据: 腾讯CDN
- 符文数据: 腾讯CDN
- 排名数据: Data Dragon（可扩展为OP.GG爬虫）

---

### ✅ 功能4：更新按钮
**位置**: 标签页4 "设置/更新"

**功能**:
- 强制更新所有数据
- 更新英雄列表
- 更新装备数据
- 更新符文数据
- 更新英雄排名
- 更新Data Dragon版本

**使用方法**:
1. 进入 "设置/更新" 标签页
2. 点击 "🔄 更新全部数据" 按钮
3. 等待更新完成
4. 查看更新状态

**更新策略**:
- 英雄列表: 1小时缓存
- 装备数据: 1小时缓存
- 符文数据: 1小时缓存
- 英雄排名: 1小时缓存
- 英雄出装: 24小时缓存

## 🎨 图标加载

程序自动从腾讯CDN加载图标：
- **英雄图标**: `https://game.gtimg.cn/images/lol/act/img/champion/{alias}.png`
- **装备图标**: `https://game.gtimg.cn/images/lol/act/img/item/{item_id}.png`
- **符文图标**: `https://game.gtimg.cn/images/lol/act/img/perk/{rune_id}.png`
- **召唤师技能图标**: `https://game.gtimg.cn/images/lol/act/img/spell/{spell_id}.png`

## ⚙️ 配置Riot API（可选）

如果需要使用功能2（查询他人信息），需要配置Riot API密钥：

### 步骤1：获取API密钥
1. 访问 https://developer.riotgames.com/
2. 点击 "Sign Up" 注册账号
3. 验证邮箱
4. 登录后，在Dashboard看到 "API KEY"
5. 复制API密钥（格式: `RGAPI-xxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx`）

### 步骤2：配置到程序
1. 启动程序
2. 进入 "设置/更新" 标签页
3. 在 "Riot API 设置" 中输入API密钥
4. 选择服务器区域
5. 点击 "保存API设置"

### 步骤3：测试
1. 进入 "查询玩家" 标签页
2. 输入召唤师名称（如 "Faker"）
3. 点击 "搜索"
4. 查看玩家信息

## 🔧 常见问题

### Q1: 程序无法启动
**A**: 检查依赖是否安装
```bash
pip install -r requirements.txt
```

### Q2: 功能2无法使用（查询他人信息）
**A**: 需要配置Riot API密钥，参见上方"配置Riot API"章节

### Q3: 图标无法显示
**A**: 检查网络连接，确保可以访问 `game.gtimg.cn`

### Q4: 数据更新失败
**A**: 检查网络连接，或手动点击 "更新全部数据" 按钮

### Q5: LOL客户端检测失败
**A**: 确保LOL客户端正在运行，并以管理员权限启动程序
